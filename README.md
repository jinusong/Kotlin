# Kotlin
## 소개
* Kotlin은 Android 공식 개발 언어입니다.
* 좀 더 가독성 뛰어나고 효율적인 코딩을 하기 위해서는 Kotlin을 배워야합니다.
* 차세대 개발자가 되기 위한 한 학생의 Kotlin Leaning 레퍼지토리입니다.
## 목차
### Start Kotlin
* 정말 Kotlin이란 언어는 무엇인가라는 주제를 중심으로 Kotlin을 탐구하는 곳입니다.
* Kotlin은 어떻게 쓰는 것인지도 함께 다루고 있습니다.
